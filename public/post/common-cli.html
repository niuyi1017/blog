前端常用命令

2022-05-10

收集整理前端常用的命令，涉及git、nginx、node、docker、linux等

git 篇

1、修改 .gitignore 文件后使其生效

git rm -r --cached .
git add .
git commit -m "update .gitignore"

2、查看git用户名信息 (设置用户信息后面跟用户信息；查看/更改全局用户信息加 --global)

git config user.name
git config user.email

node 篇

1、清除 node_modules 缓存

# npm version < 7.0.0
$ npm cache clean -f

2、查看pm2进程 

pm2 ls -a

3、查看pm2某个进程详情 

pm2 show id

4、查看pm2监控 

pm2 monit
